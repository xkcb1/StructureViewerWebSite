A software for converting the Nbt and MCA to Model

A script and software for converting the structure of NBT file and MCA file format of Minecraft into OBJ model . Use Mineways , Upx , nbtlib(2.0.4) to run the scripts . And Use Python(3.11.2) , PyQt5(5.15.9) to make the UI.

Find Out More

We've got what you need !

StructureViewer can quickly convert large NBT files into MCA files or OBJ model files, and allow users to switch material(Texture) files(.png) freely. In addition, the project also contains python scripts for rendering NBT structure off screen, python scripts for converting NBT structure files into MCA files, and scripts for parsing MCA files.
